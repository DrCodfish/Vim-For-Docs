Vim-For-Docs

Navigation

h
Move left
j
Move down
k
Move up
l
Move right
w
Jump to start of next word
b
Jump to start of previous word
0 / ^ / _
Go to beginning of line
$
Go to end of line
{ / }
Move by paragraphs
g + g
Jump to top of document
g + e
Jump backward to end of previous word
g + 0
Go to first character of the screen line
g + _
Go to last non-blank character of the line
g + j
Move down one visual (wrapped) line
g + k
Move up one visual (wrapped) line

Modes

i
Enter insert mode
a
Enter insert mode (after cursor)
v
Enter visual mode
V
Enter visual line mode
Esc
Return to normal mode
Ctrl + o
Temporary normal mode from insert mode

Editing

d + motion
Delete text over a motion
c + motion
Change text over a motion
y + motion
Yank (copy) text over a motion
p
Paste
u
Undo
r
Redo
o
Open new line below
O
Open new line above
I
Go to beginning of line & insert
A
Go to end of line & insert
